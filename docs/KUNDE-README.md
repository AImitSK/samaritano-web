## Inhalte verwalten

### Sanity CMS

Ihre Website-Inhalte werden ueber ein Content Management System (CMS) verwaltet.

**CMS-URL:** `https://[PROJEKT-ID].sanity.studio`

### Anmelden

1. Oeffnen Sie die CMS-URL
2. Melden Sie sich mit Ihrem Google-Konto an
3. Sie sehen das Dashboard mit allen Inhalten

### Seiten bearbeiten

1. Im CMS links auf "Seiten" klicken
2. Gewuenschte Seite auswaehlen
3. Aenderungen vornehmen
4. Auf "Publish" klicken

**Wichtig:** Aenderungen sind erst nach dem Publizieren auf der Website sichtbar.

### Blog-Artikel erstellen

1. Im CMS auf "Blog" oder "Artikel" klicken
2. Oben rechts "+ Erstellen" klicken
3. Titel und Inhalt eingeben
4. Bild hochladen (optional)
5. Kategorie auswaehlen
6. Publizieren

## SEO (Suchmaschinenoptimierung)

### Was Sie selbst anpassen koennen

Jede Seite hat im CMS:
- **Titel:** Wird im Browser-Tab und bei Google angezeigt (max. 60 Zeichen)
- **Beschreibung:** Erscheint in Google-Suchergebnissen (max. 160 Zeichen)
- **Bild:** Wird beim Teilen auf Social Media angezeigt

### Automatisch

- Sitemap wird automatisch erstellt
- Google kann neue Inhalte selbst finden
- Nach Aenderungen kann es einige Tage dauern bis Google aktualisiert

## Haeufige Fragen

### Wie aendere ich das Logo?

Im CMS unter "Einstellungen" → "Logo" ein neues Bild hochladen.

### Wie aendere ich die Navigation?

Im CMS unter "Navigation" die Menuepunkte anpassen.

### Wie aendere ich Kontaktdaten im Footer?

Im CMS unter "Einstellungen" die Kontaktinformationen aktualisieren.

### Warum sehe ich meine Aenderung nicht?

1. Haben Sie auf "Publish" geklickt?
2. Browser-Cache leeren (Strg+F5)
3. Einige Minuten warten
